# Talk-Type Modifiers

Each talk type adjusts which dimensions matter most and how Winston's framework adapts.

## Modifier Table

| Talk Type | Heavy Weight | De-weight | Cycling Depth | Notes |
|---|---|---|---|---|
| Lecture (50+ min) | Cycling, Engagement, Visual Aids | — | 3-5 passes | Full framework applies. Teaching branch if applicable. |
| Investor pitch (5-15 min) | Opening, Story, Closing (CTA) | Cycling | 0-1 passes | Call-to-action replaces contributions. Promise = investor hook. |
| Panel discussion | Fence, Memorability | Cycling, Visual Aids | 0-1 passes | Limited control. Focus on sharp differentiation and memorable points. |
| Virtual / Zoom | Engagement, Cycling, Visual Aids | — | 2-3 passes | Attention shorter. Engagement every 3-4 min. Slides are primary tool. |
| Lightning talk (5 min) | Opening, Memorability, Closing | Cycling | 0 passes | One clear through-line. 5 S's work BETTER here — need to be memorable fast. |
| Job talk / defense | Opening, Fence, Closing (Contributions) | — | 2-3 passes | First 5 min: vision + progress. Contributions slide stays up during Q&A. |

## How Weighting Works

- **Heavy weight**: Low scores in these dimensions get priority in the Top 3 Fixes.
- **De-weight**: Low scores are noted in the scorecard but don't count toward Top 3 Fixes unless all heavy-weighted dimensions score 4+.
- All dimensions still appear in the full scorecard regardless of weighting.

## Hybrid Types

Real talks are often hybrids (e.g., "virtual investor pitch," "lightning panel").

**Precedence rule:** Pick the **primary type** — the one that determines structure and constraints. The secondary type adds constraints but doesn't change dimension weights.

Examples:
- "Virtual investor pitch" → Primary: investor pitch (structure, CTA, no cycling). Secondary: virtual (shorter attention, more frequent engagement points, slides-first).
- "Lightning panel" → Primary: lightning (5 min, one through-line). Secondary: panel (shared stage, prepare for interruption).
- "Virtual lecture" → Primary: lecture (full framework). Secondary: virtual (can't use board, engagement every 3-4 min instead of 5).

## Framework Adaptations by Type

### Lecture / Workshop
- Full Winston framework applies
- Teaching branch: stories, questions, methods, synthesis, evaluation
- Board preferred if available
- Rules of engagement: phones away, laptops closed
- Engagement every ~5 minutes

### Investor Pitch
- Promise = "Here's why this is a $X opportunity"
- No cycling — one clear narrative arc
- Story = problem → insight → traction → ask
- Closing = call-to-action (not contributions)
- Slide rules still apply (minimal text, images)
- No board/props typically available
- "Fence" = competitive differentiation

### Panel Discussion
- Prepare 2-3 sharp points with near-miss distinctions
- Memorability critical — you're competing for attention with other panelists
- No control over tools, timing, or room
- Verbal punctuation helps audience follow your thread across interruptions
- Closing may not be in your control — front-load your contributions

### Virtual / Zoom
- Slides become primary tool (board unavailable)
- "One language processor" rule is MORE important — audience has browser tabs, email, Slack
- Cycling MORE important — higher fog-out rate
- Engagement every 3-4 minutes (not 5)
- Can't close laptops — acknowledge this reality
- Camera = your room setup. Well-lit face, clean background.
- Props can work on camera if held close

### Lightning Talk
- ONE empowerment promise, ONE through-line, ONE memorable close
- No cycling — no time. Compress fence to a single contrast.
- 5 S's work better here than for long talks — you need to be memorable in 5 minutes
- Every word counts. Hapax legomenon rule: maybe ZERO dense slides.
- Closing technique is critical — last 15 seconds are what people remember

### Job Talk / Defense
- First 5 minutes: vision (problem + your approach) + concrete progress
- Contributions slide (not conclusions) — stays up during Q&A (often 20+ min)
- Fence = how your work differs from prior art
- Practice with hostile questioners
- Situate your work: what, why, context (disciplinary, temporal, spatial)
