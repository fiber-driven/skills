# Audit Rubric — 8 Scored Dimensions

## How to Score

Read the outline as a **cold reader** — someone who was not involved in creating it. Score only what's visible in the text.

- Every score requires a **quoted evidence phrase** from the outline. No quote = max 2/5.
- Score conservatively. If evidence is weak or implicit, score lower.
- One concrete suggestion per dimension if score < 4.

### Scale

| Score | Meaning |
|---|---|
| 1 | Not present or actively harmful |
| 2 | Attempted but ineffective, or no evidence found |
| 3 | Adequate — meets basic standard |
| 4 | Strong — demonstrates clear skill |
| 5 | Exemplary — Winston would approve |

## The 8 Dimensions

### 1. Opening / Promise
Does it state what the audience gains? Is there an explicit empowerment promise ("after this you will know/be able to...")?

**Look for:** Promise statement, roadmap/menu, audience hook. **Red flags:** "Today I'll talk about..." (topic, not promise), starting with a joke, no orientation.

### 2. Structure / Cycling
Is the core idea revisited from multiple angles? Does the structure allow cycling?

**Look for:** Core thesis stated, then revisited in later sections from different angles. Verbal punctuation (numbering, progress markers, transitions). **Red flags:** Linear single-pass structure, no callbacks to thesis, sections that don't connect.

**Duration-adjusted:** Lightning talks (5 min) can't cycle — score based on whether the single pass is coherent. Lectures should cycle 3-5 times.

### 3. Fence / Near Miss
Are boundaries and contrasts explicit? Does the outline distinguish the idea from adjacent concepts?

**Look for:** "This is NOT..." statements, competitor comparisons, common misconceptions addressed, near-miss examples. **Red flags:** Idea left vague, no differentiation, audience could confuse this with something else.

### 4. Audience Engagement
Are there calibrated questions and interaction points?

**Look for:** Planned questions (not rhetorical), pause markers, audience participation moments, calibrated difficulty (not too easy, not too hard). **Red flags:** Pure monologue, no interaction, questions that are obviously rhetorical.

**Duration-adjusted:** Engagement points should appear every ~5 minutes for lectures. Lightning talks may have 0-1.

### 5. Visual Aids
Does the outline respect the "one language processor" rule? Minimal text, striking images, no bullets to be read aloud?

**Look for:** Image descriptions, "show [visual]" notes, board/prop usage, minimal slide text. **Red flags:** "Bullet points:" followed by text to read, dense tables on slides, no visual strategy, laser pointer mentioned.

### 6. Story / Narrative
Is there a clear arc with conflict and resolution?

**Look for:** Problem statement, journey/attempt, turning point or insight, result/outcome. Characters, tension, resolution. **Red flags:** Pure information dump, list of facts, no narrative thread, "and then... and then..." structure.

### 7. Memorability (5 S's)
Does the outline contain elements from Winston's Star?

**Check each:** Symbol (visual icon), Slogan (memorable phrase), Surprise (counterintuitive element), Salient idea (most memorable detail), Story (narrative journey). Score based on how many are present and how strong they are.

| Present | Score |
|---|---|
| 0-1 of 5 | 1-2 |
| 2-3 of 5 | 3 |
| 4 of 5 | 4 |
| All 5, well-executed | 5 |

### 8. Closing
Appropriate close for talk type? Delivers on promise? Memorable ending?

**Look for:** Contributions/takeaway statement, callback to opening promise, chosen end technique (joke, salute, benediction, CTA). **Red flags:** Ends with "Questions?", ends with "Thank you", no callback to promise, no clear end signal.

**Type-adjusted:** Investor pitches need a call-to-action. Job talks need a contributions slide. Lectures need a summary + end technique.

## Framework Coverage Checklist

Binary (present/absent) — separate from quality scores. Quick gap analysis.

- [ ] Empowerment promise
- [ ] Cycling (core idea revisited)
- [ ] Fence / near miss
- [ ] Verbal punctuation (structural markers)
- [ ] Calibrated questions
- [ ] 5 S's elements (any of: symbol, slogan, surprise, salient, story)
- [ ] Closing technique (joke, salute, benediction, or type-appropriate)
- [ ] "One language processor" respected (slides don't compete with speech)

### Aggregate Interpretation

| Range | Verdict |
|---|---|
| 80%+ (32-40) | Excellent — ready to deliver |
| 60-79% (24-31) | Good — targeted improvements will elevate significantly |
| 40-59% (16-23) | Needs work — structural issues to address |
| Below 40% (<16) | Major rebuild — fundamentals missing |

### Talk-Type Weighting for Top 3 Fixes

Heavy-weighted dimensions get priority in the Top 3 Fixes. De-weighted dimensions still appear in the scorecard but don't count toward the top 3 unless everything else scores 4+.

See `talk-types.md` for the full modifier table.

## After Scoring: Offer to Fix

After presenting the scorecard, ask: "Want help improving any of these sections? I can walk you through the relevant Build mode questions to strengthen specific areas."
